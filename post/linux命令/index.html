linux 常用命令

  1. 目录文件定位：ls/cd/pwd

    • ls：文件信息全掌握，基础用法ls -lah，进阶用法包括ls -lt（按修改时间排序）、ls -l | grep ^d（只显示目录）、ls -lhS（按文件大小降序排列）。

    • cd：目录切换零延迟，核心用法包括cd /etc/nginx（进入指定目录）、cd ..（退回上一级）、cd ~（回到家目录）、cd -（切换到上一次所在目录）。

    • pwd：路径确认不踩坑，显示完整绝对路径，避免路径错误。

  2. 文件管理：touch/mkdir/rm

    • touch：快速创建文件，基础用法touch nginx.conf，进阶用法包括touch -d “2025-12-01” test.txt（创建指定修改时间的文件）。

    • mkdir：目录创建省时间，单目录创建mkdir /data/backup，多级目录创建mkdir -p /data/{logs,conf,backup}，设置权限mkdir -m 755 docs。

    • rm：删除操作需谨慎，删除文件rm -f old.log，删除目录rm -rf redundant_dir，安全技巧包括删除前确认内容、备份重要文件、避免在根目录执行危险命令。

  3. 系统监控：free/df/top

    • free：内存状态秒掌握，基础用法free -h，进阶用法包括free -s 5（每隔5秒刷新）、free -m（以MB为单位显示）。

    • df：磁盘空间早预警，基础用法df -h，进阶用法包括df -i（查看inode使用情况）、df -h /data（查看特定分区）。

    • top：进程监控全视角，基础操作包括查看CPU和内存占用、终止异常进程，进阶用法包括top -p 1234（监控特定进程）、top -n 1 -b（非交互式输出）。

  4. 远程运维：ssh

    • 密码登录：基础用法ssh root@192.168.1.100，端口自定义ssh admin@10.0.0.5 -p 2222。

    • 密钥登录：步骤包括生成密钥对、上传公钥、配置快捷登录，进阶技巧包括ssh代理转发，避坑提醒包括权限设置。

  5. 运维效率技巧

    • 命令输入技巧：按Ctrl+U清空当前输入，Ctrl+K删除光标后的内容，Ctrl+W删除光标前的一个单词。

    • 历史命令复用：使用history查看历史命令，!<编号>执行特定命令，Ctrl+R搜索历史命令。

    • 命令参数查询：使用command –help查看简要参数，man ls查看完整手册，info top查看详细用法。

    • 长命令保存：将常用复杂命令写入脚本，赋予执行权限后直接运行。

    • 命令执行不中断：使用nohup command &避免断开ssh导致任务终止。

通过熟练掌握这些核心命令和技巧，运维人员可以高效应对日常工作中的大部分场景，提高工作效率并降低操作风险。
