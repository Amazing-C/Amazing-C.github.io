正则表达式

相关符号含义理解

  • [] 表示 其中的内容有一个满足条件就行
  • ^ 表示 字符串开头 在[]内部的 ^ 表示非
  • $ 表示 字符串结尾
  • . 表示一个任意字符 除了换行符号\n
  • * 表示前面字符0个或多个字符
  • + 表示前面有1个或以上满足
  • ? 表示前面字符的可以有也可以没有
  • \d 表示纯数字
  • \B 表示 非单词边界 两边必须是同种类型的 常用于表示存在 内部的子字符串 eg. helloworld 用 \Brld\B 查找不到
  • \b 表示单词边界 常用于匹配子字符串，一侧必须被非单词包裹 或为空 ，另一侧为单词，单词(包括_)与非单词的夹缝 eg. ,helloworld, \bhelloworld\b 可以查到 但是 如果字符串是 hhelloworldd \bhelloworld\b 就什么都差不到
  • {} 表示 其中的内容表示前面字符出现的次数 eg. {n} 前面的字符N次 ,{2,} 前面的字符 2次以上
  • () 表示 子匹配项 可以把一个很长的需求 分解为多个小需求 然后 分别匹配需求 括号中间可以写 正则表达式 （）匹配到的结果会存储起来 “捕获组” (?:xxx) 这个不会存储
  • \w :匹配数字 字母 下划线 = [A-Za-z0-9 _]
  • \s 匹配空白字符 （空格、制表符\t、换行符\n等）
  • \W 表示非\w = [^A-Za-z0-9]
  • \S表示非\s
  • \D 表示非\d

相关概念

  • 贪婪匹配：尽可能匹配的长，eg. a1bcccccbc a.*b → a1bcccccb
  • 非贪婪匹配：尽可能匹配的短，eg. a1bcccccbc a.?b → a1b
  • 零宽断言：不匹配具体字符，只判断 “位置” 是否满足条件（“零宽” 指不占用字符长度）

正则表达式使用

python

re 模块

flags--flags修饰符：用来改正则表达式的匹配行为 eg. 忽略大小写 匹配多行 正则表达式可加空格 注释之类

  1. re.match(pattern, string, flags=0)
    • 匹配字符串的开头
    • 是否string开头 满足 pattern 就返回Match对象，不满足就返回None
    • 应用：手机号开头匹配 身份证开头匹配
  2. re.search(pattern, string, flags=0)
    • 整个字符串
    • 是否string 中有满足 pattern，只要有一个， 就返回Match对象，没有就返回None
    • 应用：查询是否有关键词
  3. re.findall(pattern, string, flags=0)
    • 整个字符串
    • 是否string 中有满足 pattern，满足就返回一个列表
    • 应用：查询所有满足的关键词
  4. re.finditer(pattern, string, flags=0)
    • 整个字符串
    • 是否string 中有满足 pattern，满足就返回一个迭代器
    • 应用：查询所有满足的关键词的迭代器
  5. re.sub(pattern, repl, string, count=0, flags=0)
    • 整个字符串
    • 如果string 中有满足 pattern，满足就替换成repl，返回替换后的字符串
    • 应用：替换字符串
  6. re.split(pattern, string, maxsplit=0, flags=0)
    • 整个字符串
    • 如果string 中有满足 pattern，满足分割字符串，返回分割后的字符串的列表
    • 应用：分割字符串
  7. re.compile(pattern, flags=0)
    • 预编译正则表达式
    • 返回Pattern对象
    • 应用：多次使用同一正则表达式的场景,提高效率
    • eg. pattern = re.compile(r’\d+’) match = pattern.match(‘123456’)

javaScript

正则表达式在JavaScript 中 内置 下面是两种定义方法

  1. 构造函数：const regex = new RegExp(pattern, flags)

  2. 字面量：const regex = /pattern/flags

注意：python 和 javaScript 的正则表达式语法不同

- python 内置的字符串不支持 正则表达式 作为输入参数 需要使用re模块 例如str.replace(old,new,count)。如果需要使用正则表达式来替换字符串（例如替换所有数字），需要使用re.sub()
- js 字符串内置了正则表达式 例如str.replace(regex, replacement) 可以使用正则表达式作为参数,同时上面的 const regex = /pattern/flags 也可以使用 regex.test(str) regex.exec(str) 这种方法
- 注意 js中 const regex = /pattern/flags 这种模式需要用/ / 包裹正则表达式 同时后面需要添加 g i 之类的修饰,而通过 new RegExp(pattern, flags) 这种模式则不需要 但是区别转义字符 还是需要用 \\d+ 类似于 python 中的 r'\d+'
